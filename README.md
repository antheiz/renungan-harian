Nah, orang Kristen, bukankah ini menggambarkan penderitaan-penderitaan yang sedang engkau alami? Bukankah Allah sedang memperjelas anugerahmu, dan mempertumbuhkannya? Bukankah ini alasan Dia berperkara dengan engkau?
